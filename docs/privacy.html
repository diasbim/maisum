Politica de Privacidade

Ultima actualizacao: 12 de maio de 2026

Esta politica explica como o MaisUm recolhe, usa e protege informacao quando usas o nosso site e a app de fidelizacao.

1. Dados que recolhemos

  • Dados da barbearia ou salao: nome do negocio, contacto, endereco e detalhes de configuracao.
  • Dados de clientes: nome, telefone, historico de vendas e pontos.
  • Dados de transacoes: valores, datas, metodo de pagamento e recompensas.
  • Mensagens e eventos de WhatsApp ligados ao fluxo de fidelizacao.
  • Dados tecnicos: modelo do dispositivo, sistema operativo, registos de erro e identificadores tecnicos.
  • Dados de navegacao no site para entender interesse e melhorar a experiencia.

2. Base legal e legislacao de Mocambique

Tratamos dados de acordo com a legislacao de protecao de dados pessoais de Mocambique. As bases legais podem incluir a execucao do contrato, cumprimento de obrigacoes legais, interesses legitimos e consentimento, quando aplicavel.

3. Como usamos os dados

  • Para registar vendas, atribuir pontos e enviar notificacoes aos clientes.
  • Para prestar suporte, responder pedidos e melhorar o produto.
  • Para garantir seguranca, prevenir abuso e manter a qualidade do servico.
  • Para analisar uso geral do site e da app (sem vender dados).

4. Partilha de dados

Podemos partilhar dados com fornecedores que nos ajudam a operar o servico (alojamento, analitica, mensageria). Esses parceiros so usam a informacao para esse fim. Nao vendemos dados pessoais.

5. Armazenamento e seguranca

Aplicamos medidas tecnicas e organizacionais para proteger os dados. Quando a internet falha, dados podem ficar guardados no dispositivo e sincronizam quando a ligacao volta.

6. Retencao

Guardamos dados enquanto a tua conta estiver ativa e pelo tempo necessario para cumprir o servico. Podes pedir remocao a qualquer momento.

7. Os teus direitos

De acordo com a legislacao de Mocambique, podes pedir acesso, correcao, atualizacao, eliminacao ou oposicao ao tratamento dos teus dados. Basta contactar-nos pelo WhatsApp.

8. Contacto

Para questoes de privacidade, fala connosco em +258 82 326 2347.

9. Alteracoes

Podemos atualizar esta politica para refletir melhorias no servico. Quando isso acontecer, vamos atualizar a data acima.
